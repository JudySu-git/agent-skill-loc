# General PQF Forms

This folder contains CSV data files for Avetta PQF (Pre-Qualification Form) topic
forms — the core modular questionnaires that make up the Avetta supplier
prequalification system. These forms are assigned to suppliers based on their
work types, geographic locations, and client requirements.

Topics covered include:
- Safety programs (General Safety, HSE, Safety Management Systems, PSM)
- Corporate responsibility (Labor, Environment, Code of Conduct, Modern Slavery)
- Sustainability and ESG (GHG Emissions, Carbon Reduction, Sustainability Stage 1)
- Regulatory compliance (DOT/PHMSA, BSEE, MSHA, IIPP/CalOSHA, TWIC)
- Supplier diversity (HUBZone, Veteran-Owned, Women-Owned, LGBTQ+, Disability-Owned)
- Transportation and vehicle safety (Canada, Colombia, Australia, DOT)
- Drug & alcohol programs (US, Canada, UK/global)
- Country/region-specific supply chain compliance (SAPIN2, Germany Supply Chain Act, Guatemala OHS)

## CSV File Structure

Each CSV file contains the following columns:
- `_id`: MongoDB document ID
- `mongoId`: MongoDB reference ID
- `avettaId`: Avetta internal ID (matches formDisplayId for the top-level form group)
- `mongoObject`: Object type — `questionGroup` or `question`
- `field`: Field type — `questionGroupText`, `questionText`, `description`, `responseOption*`
- `formDisplayId`: The form's display ID (matches the formID in the filename)
- `questionGroupType`: Always `form` for top-level entries
- `en`: English text content
