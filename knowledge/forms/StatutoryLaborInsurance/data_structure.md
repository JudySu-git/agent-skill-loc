# Statutory Labor Insurance Forms

This folder contains CSV data files for Avetta forms related to statutory workers'
compensation, labor insurance, and government-mandated employer insurance schemes.
Forms cover:
- Canadian Workers' Compensation Board (WCB) rate statements and notices
  (Manitoba, Alberta, BC, Saskatchewan, PEI, NS, NB, Yukon, NWT/Nunavut, Ontario, Quebec)
- U.S. state-specific workers' compensation (Ohio, Wyoming, Washington, North Dakota)
- New Zealand ACC (Accident Compensation Corporation)
- FRA (Federal Railroad Administration) drug & alcohol program compliance
- CNESST (Quebec), WSCC (NWT/Nunavut), WorkSafeBC reports

## CSV File Structure

Each CSV file contains the following columns:
- `_id`: MongoDB document ID
- `mongoId`: MongoDB reference ID
- `avettaId`: Avetta internal ID (matches formDisplayId for the top-level form group)
- `mongoObject`: Object type — `questionGroup` or `question`
- `field`: Field type — `questionGroupText`, `questionText`, `description`, `responseOption*`
- `formDisplayId`: The form's display ID (matches the formID in the filename)
- `questionGroupType`: Always `form` for top-level entries
- `en`: English text content
